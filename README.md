# smart-expense-tracker
Developed a web-based expense tracking system using Python, Flask, and SQLite that allows users to record and manage daily expenses. Implemented interactive pie chart visualization using Chart.js to analyze spending by category.

Modern Flask web app for tracking expenses with charts & analytics!

## ✨ Features
- ✅ Add expenses (Category + Amount + Date)
- 📊 Category pie charts
- 📈 Monthly trend analysis
- 💡 Smart insights
- 📥 Complete CSV reports

## 🚀 Quick Start
```bash
pip install flask
python app.py
